# 爬虫示例包

文件结构

```
---- data   存放数据
---- example 爬虫例子
```

使用说明:

```
go get -u -v github.com/hunterhug/spiderexample
```

## 一.投资界爬虫pedaily

1. companysearch.exe可通过关键字查找一家机构的简单信息
2. companytouzi.exe可通过公司代号查找一家机构的投资情况
